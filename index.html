Free Firebase Setup

5 minutes, 100% free, no credit card.

  1. Go to console.firebase.google.com and sign in with Google.
  2. Click Add project. Name it "Goal Tracker". Skip Google Analytics.
  3. Once it's ready, click the </> web icon on the project page. Give it a nickname, click Register.
  4. Copy the firebaseConfig object that shows up. Paste the whole {...} block into the box on the previous screen.
  5. In the left sidebar, click Build → Realtime Database. Click Create Database. Pick "Start in test mode" (you can lock it later). Click Enable.
  6. Come back here, paste the config, hit Connect Sync. Repeat on Chardae's phone with the same config. Done!

Need the databaseURL? It's on the Realtime Database page, looks like https://your-project-default-rtdb.firebaseio.com. Add it to the config you pasted.
